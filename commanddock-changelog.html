v1.0.0

February 18, 2026

Features

  • ⚡ Auto-Detection — scans workspace and detects Git, Node.js, Docker, Python, Rust, Go, Java, Flutter, Tailwind, Firebase, Turborepo, and 20+ stacks
  • 🎯 One-Click Execution — run commands directly via Electron's child_process
  • 🔧 Custom Variables — {{branch}}, {{host}}, {{user}} with live preview in ConfigPanel
  • ⌨️ Command Palette — Ctrl+K fuzzy search across all commands
  • 📂 Smart Workspace — drag & drop projects, directory browser, manual path entry, re-scan
  • ⭐ Favorites & Shortcuts — star commands, run with Ctrl+1-9
  • 🌐 Multilingual — built-in Spanish and English with instant switching
  • 📤 Export / Import — full config backup as JSON
  • 🛡️ Dry Run Mode — preview commands without executing
  • 📝 Open in Editor / Terminal — quick project access buttons
  • ⚡ Quick Actions — top 4 most-used commands from history
  • 📜 History Panel — recent command execution log
  • 🛠 Custom Commands — add your own commands to any project
  • 🔔 OS Notifications — notifies when long commands complete
  • 💬 Feedback Modal — Bug/Feature/Other reports via Google Forms
  • ☕ Ko-fi Support — donation link in footer
