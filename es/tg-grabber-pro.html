Extensión Chrome Gratuita

TG Media Grabber Pro

El descargador definitivo para Telegram Web. Fotos, videos y GIFs — masivamente, con un clic.

Añadir a Chrome — Gratis Ver en GitHub

Funciona en Telegram Web K y A · Chrome, Edge, Brave · Changelog

Repleto de Funciones

📦

Descarga Masiva

Escanea chats y canales completos. Descarga cientos de fotos y videos con un solo clic.

🖼️

Vista Previa

Navega por todo el contenido detectado en una galería visual antes de descargar. Elige exactamente lo que quieres.

🧠

Duplicados Inteligentes

Detección inteligente de duplicados por nombre y ID de mensaje. Nunca descargues el mismo archivo dos veces.

⚡

Ultra Rápido

Descargas directas a través de la API de Telegram para máxima velocidad, sin servidores externos.

🔓

Contenido Restringido

Evita las restricciones de reenvío para guardar contenido de canales que bloquean las descargas.

🎯

Filtrado por Tipo

Filtra por Fotos, Videos o GIFs. Activa y desactiva tipos para descargar solo lo que importa.

📏

Límites de Tamaño

Establece un tamaño máximo para omitir videos pesados. Elige desde 500 MB hasta 4 GB o sin límite.

🛡️

Descargas Resilientes

Recuperación elegante de errores para mantener la cola en movimiento. Los fallos no detendrán tu lote.

📁

Archivos Organizados

Archivos guardados en TG_Media/{NombreChat}/. Todo perfectamente organizado por conversación.

Cómo Funciona

1

Instalar

Añade TG Grabber Pro a Chrome desde la Web Store. Funciona al instante — cero configuración.

2

Escanear

Abre cualquier chat en web.telegram.org. Haz clic en el icono de la extensión y pulsa Scan.

3

Descargar

Previsualiza en la Galería o haz clic en Download All. Los archivos se guardan organizados por nombre de chat.

Instalación desde GitHub (opcional)

Si prefieres instalar la extensión manualmente, descarga el ZIP desde las releases de GitHub. Luego en Chrome: chrome://extensions → Modo desarrollador → Cargar descomprimida → selecciona la carpeta del ZIP.
