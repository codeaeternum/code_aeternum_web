v1.0.1

18 de febrero de 2026

Mejoras

  • 🖥️ Compilación Multi-Plataforma — ahora disponible para Windows (NSIS + portable), macOS (DMG) y Linux (AppImage + deb)
  • 🎨 Ícono Personalizado — nuevo ícono de marca para todas las plataformas
  • 🔄 Pipeline CI/CD — GitHub Actions compila automáticamente en todas las plataformas al crear un tag
  • 🏷️ App ID Actualizado — com.codeaeternum.commanddock
🎉 Lanzamiento Inicial

v1.0.0

18 de febrero de 2026

Funcionalidades

  • ⚡ Auto-Detección — escanea tu workspace y detecta Git, Node.js, Docker, Python, Rust, Go, Java, Flutter, Tailwind, Firebase, Turborepo y más de 20 stacks
  • 🎯 Ejecución con un Clic — ejecuta comandos directamente vía child_process de Electron
  • 🔧 Variables Personalizadas — {{branch}}, {{host}}, {{user}} con vista previa en vivo
  • ⌨️ Command Palette — Ctrl+K búsqueda fuzzy entre todos los comandos
  • 📂 Workspace Inteligente — arrastra carpetas, explora directorios o escribe la ruta manualmente
  • ⭐ Favoritos y Atajos — marca comandos con estrella, ejecútalos con Ctrl+1-9
  • 🌐 Multiidioma — español e inglés con cambio instantáneo
  • 📤 Exportar / Importar — respaldo completo de configuración como JSON
  • 🛡️ Modo Dry Run — previsualiza comandos sin ejecutarlos
  • 📝 Abrir en Editor / Terminal — botones de acceso rápido al proyecto
  • ⚡ Acciones Rápidas — los 4 comandos más usados del historial
  • 📜 Panel de Historial — registro de ejecuciones recientes
  • 🛠 Comandos Personalizados — agrega tus propios comandos
  • 🔔 Notificaciones del SO — avisa cuando terminan comandos largos
  • 💬 Modal de Feedback — reportes Bug/Feature/Otro vía Google Forms
  • ☕ Ko-fi — enlace de donación en el footer
