Packed with Features

📦

Bulk Download

Scan entire chats and channels. Download hundreds of photos and videos with a single click.

🖼️

Gallery Preview

Browse all detected media in a visual gallery before downloading. Select exactly what you want.

🧠

Smart Duplicates

Intelligent duplicate detection by filename and message ID. Never download the same file twice.

⚡

Lightning Fast

Downloads directly through Telegram's API for maximum speed. No external servers involved.

🔓

Restricted Content

Bypass forwarding restrictions to save media from channels that block downloads.

🎯

Type Filtering

Filter by Photos, Videos, or GIFs. Toggle types on and off to download only what matters.

📏

Size Limits

Set a max file size to auto-skip oversized videos. Choose from 500 MB to 4 GB or no limit.

Resilient Downloads

Graceful error recovery keeps the queue moving. Failures won't stall your entire batch.

Organized Files

Files saved in TG_Media/{ChatName}/. Everything neatly organized by conversation.

How It Works

1

Install

Add TG Grabber Pro from the Chrome Web Store. No setup required — it works as soon as you install.

2

Scan

Open any Telegram chat on web.telegram.org. Click the extension icon and hit Scan.

3

Download

Preview in the Gallery or click Download All. Files are saved organized by chat name.

Manual install (without the Store)

Prefer to install without the Chrome Web Store? Download the ZIP from GitHub releases. Then in Chrome: chrome://extensions → turn on Developer mode → Load unpacked → select the extracted folder.
